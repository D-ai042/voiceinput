# VoiceInput - 语音输入小助手（Node.js 版本）

一个轻量的桌面语音输入工具，点击说话，自动转文字填入任意文本框。

## 功能特性

- 点击悬浮按钮开始录音
- 录音结束后自动发送到百度语音识别 API
- 识别结果自动填入当前聚焦的文本框
- 支持 Windows 桌面
- 始终置顶的悬浮窗
- 可拖拽定位
- 系统托盘支持

## 安装依赖

```bash
cd node-voice-input
npm install
```

## 首次配置

1. 访问 [百度智能云控制台](https://console.bce.baidu.com/)
2. 搜索"语音技术"，创建应用
3. 获取 App ID、API Key、Secret Key
4. 运行程序后点击悬浮窗右上角设置，填入上述密钥

## 运行程序

```bash
npm start
```

## 打包发布

```bash
npm run build
```

打包后的文件在 `dist\VoiceInput.exe`

## 使用方法

1. 启动程序，悬浮球出现在桌面右下角
2. 将光标放在任意文本框
3. 点击悬浮球开始录音
4. 再次点击结束录音
5. 识别结果自动填入文本框

## 快捷操作

| 操作 | 说明 |
|------|------|
| 单击 | 开始/结束录音 |
| 右键 | 菜单（设置、退出） |
| 拖拽 | 移动窗口位置 |

## 技术栈

- Electron - 桌面 GUI
- Web Audio API / MediaRecorder - 录音
- 百度语音识别 API - 长音频转文字
- RobotJS - 模拟键盘输入

## 常见问题

### 百度 API 调用失败
- 检查 API 密钥是否正确
- 检查网络是否正常
- 检查百度云账户是否欠费

### 文字没有填入文本框
- 检查是否有其他程序占用了剪贴板
- 确保目标文本框获得焦点
